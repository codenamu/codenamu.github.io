--- layout: page banner: codeforseoul.png title: Code for Seoul --- 코드포서울은 기술을 통해 사회적 문제 해결을 위한 코드나무의 프로젝트 그룹입니다.

IT 기술을 통하여 우리가 살고 있는 도시에 존재하는 많은 문제들과 불편함을 해결하기 위해 노력합니다. IT기술을 가지고 개발자와 디자이너 활동가들이 모여 서울에서 발생하는 사회적 문제를 고민하고 IT 기술로 해결할 수 있는 방법을 고민합니다. 그리고 우리가 가진 기술을 가지고 직접 사회적 문제를 해결하기 위하여 결과물을 만들어갑니다.

Code for Seoul with Code for All

2010년 이후로 열린 정부와 관련된 스터디를 시작으로 코드나무는 그동안 번역, 공공데이터 해커톤, 보고서 / 연구자료 작성 등 시민들이 정부 활동에 적극적으로 참여할 수 있도록 하기 위한 열린 정부 활동을 꾸준히 진행해왔습니다.

국내에서의 다양한 프로젝트를 바탕으로 코드나무는 함께 열린 정부를 위해 노력하는 해외 조직 및 기관들과의 협력을 강화하기 위하여 Code for Amercia Brigade 네트워크에 합류하였습니다. 이로써 전 세계적으로 각 도시를 기반으로 사회적 문제를 해결하기 위한 커뮤니티들과의 협력을 강화하고 소통하며 함께 고민할 수 있는 네트워크에 Code for Seoul이라는 이름으로 합류하였습니다.

지금껏 지속해왔던 국내 활동들 뿐만 아니라 해외의 좋은 사례들도 함께 배울 수 있는 좋은 기회로서, 그리고 실질적으로 서울을 위한 프로젝트를 진행하고 만들어내는 워킹 그룹으로서 보다 나은 서울을 위해 고민하고 행동하는 워킹그룹이 될 것입니다.
